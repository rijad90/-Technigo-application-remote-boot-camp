What is Scrum?

Scrum is a lightweight framework that helps people, teams and organizations generate value through adaptive solutions for complex problems.

Scrum is about continuously shipping value to customers. It's a framework for getting work done. All kinds of teams use Scrum, HR, marketing, design and more.

The most common is software development, it's a great framework for many development teams for a few reasons.


With software, requirement change, goals change, situations change. Scrum embraces those changes

Scrum consists of several sprints. You break down big, complex projects into small pieces making projects more manageable.
